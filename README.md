# Muazom — Personal Portfolio

A modern, mobile-responsive personal portfolio website built with **HTML**, **Tailwind CSS**, **GSAP**, and **vanilla JavaScript**.  
Designed with a mature dark-blue aesthetic to showcase my work in **web development** and **cybersecurity**.

---

## Overview

This portfolio represents my journey as a student developer with strong interests in:
- Cybersecurity & ethical hacking
- Secure web development
- Clean UI/UX design
- Performance-focused frontend builds

The site uses a glassmorphism design, smooth animations, and a minimal professional layout.

---

## Tech Stack

- **HTML5**
- **Tailwind CSS**
- **JavaScript**
- **GSAP (GreenSock Animation Platform)**

No frameworks. No React. Pure and lightweight.

---

## Features

- Fully responsive (mobile, tablet, desktop)
- Dark blue & white professional theme
- Glassmorphism UI cards
- Smooth scroll-based animations
- Simple structure, easy to edit
- Fast loading and lightweight
